# Répartition des Interviews - Validation StageComplete

**Objectif** : 15 interviews en 3 jours (mardi-jeudi)
**Durée par interview** : 10-15 min max
**Méthode** : Appels téléphoniques

---

## 📊 Répartition Proposée

### TOI (Seydou) - 8 interviews
- **5 Venues** (bars, cafés-concerts, théâtres)
- **3 Artistes** (musiciens, comédiens)

### ELLE - 7 interviews
- **5 Venues** (bars, cafés-concerts, théâtres)
- **2 Artistes** (musiciens, comédiens)

**Pourquoi cette répartition ?**
- Plus de venues car c'est le côté qui PAIE (priorité validation)
- Tu fais plus d'artistes car tu connais mieux le produit
- Elle fait plus de venues car elle connaît mieux le marketing B2B

## 📞 Script d'Interview (Pour Elle)

**Tu lui partages ce script exact** :

### INTRODUCTION (1 min)

```
"Bonjour, c'est [ton prénom] de StageComplete.

Je fais une étude de marché sur le booking d'artistes.
J'aimerais comprendre comment vous gérez ça aujourd'hui.

Pas de vente, juste 10 minutes de questions.
Vous avez le temps maintenant ?"

[Si non] "Quel créneau vous conviendrait mieux ?"
```

### QUESTIONS VENUES (8 min)

**BLOC 1 : Contexte (3 min)**

1. **"Vous programmez combien de concerts ou spectacles par mois en moyenne ?"**
   → Noter : [X] par mois

2. **"Comment vous trouvez vos artistes actuellement ?"**
   → Noter : [Agence / Réseau personnel / Réseaux sociaux / Autre]

3. **"Quelle est votre plus grosse difficulté avec ce système actuel ?"**
   → **ÉCOUTER ATTENTIVEMENT. Noter mot pour mot.**

**BLOC 2 : Validation Douleur (3 min)**

4. **"Sur une échelle de 1 à 10, à quel point c'est pénible ?"**
   → Noter le chiffre

5. **"Vous avez déjà essayé d'autres solutions ? Lesquelles ?"**
   → Noter (agences, plateformes, etc.)

6. **"Si une plateforme réglait [répéter leur problème principal], avec 500+ artistes vérifiés et 0% de commission, combien seriez-vous prêt à payer par mois ?"**
   → Noter le montant (ou "rien" si gratuit seulement)

**BLOC 3 : Conversion (2 min)**

7. **"On lance une version beta dans 2 semaines. Si je vous donne un accès gratuit pour tester pendant 30 jours, vous seriez partant ?"**
   → Oui / Non / Peut-être

8. **[Si oui ou peut-être] "Super ! Je prends votre email pour vous recontacter."**
   → Noter email

**CLÔTURE (1 min)**

```
"Merci infiniment pour votre temps !

Une dernière chose : vous connaissez d'autres gérants
de bars ou théâtres qui auraient ce même problème ?"

[Noter 1-2 références si possible]

"Merci encore, excellente journée !"
```

### QUESTIONS ARTISTES (8 min)

**Même structure, questions adaptées :**

1. "Combien de dates vous faites par mois actuellement ?"
2. "Comment les venues vous trouvent aujourd'hui ?"
3. "Quelle est votre galère principale pour vous faire connaître ?"
4. "Sur 10, à quel point c'est frustrant ?"
5. "Vous avez essayé quoi comme solutions ?"
6. "Vous paieriez combien/mois pour un profil pro en ligne avec URL personnalisée ?"
7. "Je lance une beta gratuite, vous testeriez ?"
8. [Si oui] Email pour recontact

## ✅ Après Chaque Interview

**Procédure pour ELLE et TOI :**

1. **Remplir le Google Sheet immédiatement** (2 min)
   - Toutes les réponses
   - Score problème /10
   - Montant willing to pay
   - Accepte beta (Oui/Non/Peut-être)

2. **Noter 2-3 citations clés** (1 min)
   - Les phrases exactes du prospect
   - On les réutilisera pour les pubs

3. **Pause 5 min** (important)
   - Respirer
   - Se détendre
   - Préparer le prochain appel

4. **Prochain appel**

---

## 🎯 Critères de Succès (Pour l'Analyse Vendredi)

**On considère que c'est validé SI :**

✅ **10+ prospects** (sur 15) disent que le problème est fort (score 7+/10)
✅ **8+ prospects** (sur 15) sont prêts à payer >0€
✅ **8+ prospects** (sur 15) acceptent de tester la beta

**SI ces 3 critères sont atteints → GO pour les campagnes pub**
**SINON → On itère ou on pivote**

## 💡 Tips pour Elle (Partage ces conseils)

### Avant les Appels
- ☕ Prends un café, sois détendue
- 📝 Aie le script sous les yeux
- 🎧 Utilise des écouteurs (meilleures mains libres)
- ⏱️ Met un timer 15 min (ça rassure)

### Pendant les Appels
- 🎤 **Écoute 80% du temps, parle 20%**
- 🤐 Ne défends pas le produit, juste ÉCOUTE
- ❓ Si la personne dévie, ramène-la gentiment : "Super intéressant ! Et du coup sur [question], vous pensez quoi ?"
- 😊 Souris en parlant (ça s'entend au téléphone)

### Après les Appels
- 📊 Remplis le Google Sheet TOUT DE SUITE (sinon tu oublieras)
- 🎉 Célèbre après chaque appel (petit fist pump, c'est fait !)
- 😌 C'est OK si quelqu'un dit non ou raccroche

### Si Quelqu'un Refuse
**C'est une DATA PRÉCIEUSE** → Note pourquoi
- "Pas le temps" → Pas assez douloureux
- "Pas intéressé" → Mauvais prospect ou problème pas fort
- "Déjà une solution" → Concurrent, note lequel
