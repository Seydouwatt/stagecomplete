Étapes de test :

  1. Aller sur /browse (page avec barre de recherche en haut)
    → Ouvrir /browse
  2. Faire une recherche
    - Taper un terme de recherche (ex: "jazz")

  3. Ouvrir les filtres
    - Cliquer sur "Filtres"
    - Sélectionner un genre (ex: "Rock")
    - ✅ RÉSULTAT ATTENDU: Rester sur /browse avec les paramètres ajoutés
  4. Ajouter une localisation
    - Dans la barre de recherche, ajouter une ville (ex: "Paris")
    - ✅ RÉSULTAT ATTENDU: Rester sur /browse avec ?q=jazz&location=Paris
  5. Vérifier depuis la page d'accueil
    → Ouvrir page d'accueil
    - Faire une recherche depuis la page d'accueil
    - ✅ RÉSULTAT ATTENDU: Navigation vers /search (comportement normal)

Vérifications :

  • ☐ La recherche depuis /browse reste sur /browse
